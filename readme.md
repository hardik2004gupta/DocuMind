# 🧠 DocuMind

### *Chat with your documents using AI*

> DocuMind is an AI-powered Retrieval-Augmented Generation (RAG)
> application that enables users to upload documents and perform
> context-aware question answering using Groq LLM.

## 🧠 How It Works

1.  Upload PDFs\
2.  Split into chunks\
3.  Generate embeddings\
4.  Store in FAISS\
5.  Retrieve relevant chunks\
6.  Generate answer via Groq
